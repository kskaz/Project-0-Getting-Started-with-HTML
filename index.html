Notes for Basics of the Web and HTML

Basics of the Web

The web is a collection of documents which are made out of HTML. HTML stands for Hypertext Markup Language.
HTML glues everything (text, videos, images, etc) together. Documents can contain links to other HTML documents.
This interconnection of documents gives the Web its name.

Major Components of the Web

Your computer uses an application called a browser to view HTML documents that reside on other servers on the Internet.
It uses the HTTP protocol to send and receive this information.

Composition of HTML

HTML is made up of:
- text content (what you see)
- markup (determines what it looks like)
- references to other documents (images, videos, etc)
- links to other HTML pages

HTML uses tags to change the appearance of content in a document. They start with an opening tag and end with the closing tag.
The closing tag shares the same name as the opening tag but also has a slash in front of the tag's name.
The content being modified by the tag lies between the opening and closing tag.

Some tags we learned about:
bold
italics

HTML attributes

HTML attributes are tags that include attributes within them. These attributes have values.

Examples are:

anchor tag - links to another page
attribute: href
value: url of page

image tag - displays an image on the page

attribute: src
value: location of image
attribute: alt
value: text if image doesn't appear

Some tags do not have closing tags if there is no content - the image tag is one of these.

Whitespace:

In HTML, all whitespace turns into a single space. We use tags to control how whitespace appears on our page.

Example tags:
br - line break (void tag)
p - paragraph - has opening and closing tag - content between tags will be contained in the paragraph

Block vs Inline

Inline tags are just text. Block tags draw invisible boxes around its content and can have a height and width.

HTML structure

A complete HTML document includes:
doctype: type of document
html tag: at beginning and end of document
head tag: contains title tag and other meta data (CSS and JavaScript)
body tag: contains document content

We are human but computers are not...

Computers can only do what we ask them to do. If we make a mistake in our programming, unexpected results may occur.
If we forget how to do something, we know we can always look it up. It's impossible to remember everything.
